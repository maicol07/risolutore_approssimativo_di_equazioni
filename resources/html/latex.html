Informazioni sul formato LaTex

Il formato LaTex è un linguaggio di markup per la preparazione di testi ed ha una rigida sintassi che puoi visionare qui.

Tuttavia, si consiglia di utilizzare un editor visivo e poi copiare e incollare il risultato, come MathLive o iMathEQ.

Nel primo, basta inserire l'equazione nella casella di testo e premere il pulsante Show code section nella barra laterale. Il codice LaTex verrà mostrato nel box scuro in fondo alla pagina.

Nel secondo, invece, occorre inserire l'equazione nell'area di testo in alto e si riceverà il codice LaTex nel box denominato LaTex dopo aver premuto il pulsante Get LaTex.
